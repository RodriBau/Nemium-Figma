Conceptos escenciales Del Desarrollo Web

1. Que es HTML

Hyper Text Markup Language o Lenguaje de marcas Hipertexto. Es el componente mas basico de la Web. Define el significado
y estructura dle contenido.



2. Etiquetas

Las etiquetas HTML van a ayudarnos a brindar una estrutura y semantica al contenido de nuestro diseño website y cada una tiene
caracteristicas y usos diferentes aunque visualmente den un resultado similar.

Muchas etiquetas son facil de diferencias como los headins o las imagenes y otras diferentes como header, footer, section o
div, aqui es donde la semantica sera la respuesta y si sabemos un poco de ingles tambien podremos darnos una pequeña idea
de su uso pero en este curos vamos a usar las mas importantes y que seran parte de tu dia a dia como Sofware Developer.

3. Doom

El Document Object Model es una estructura de árbol que representará todos nuestros proyectos web como si un árbol
genealógico fuera: padres, hijos, hermanos con niveles infinitos. cuando entendemos esta anidación podemos identificar
dependencias, herencias en css y que tan complejo es nuestro proyecto.



4. Semantica

La semántica le brinda sentido a cada elemento existen en un sitio web, algunos elementos será meramente decorativos y no
deben significar nada pero otros serán títulos y deberán estar en diferente jerarquías o alguna secciones serán más relevantes
que otras, esto ayudará a los motores de búsqueda como Google o Duck Duck Go a diferenciar tu contenido, categorizar y será
la herramientas más valiosa para estar en los primeros resultados de búsqueda SEO. También ayudará a la accesibilidad de tu
sitio web, para que personas con habilidades diferentes puedan entender cada contenido.

5. Atributos

Los atributos le dan características extra a las etiquetas para complementar información y son pre programadas por el
navegador, algunos atributos funcionan en todas las etiquetas de la misma forma como ‘class’ pero otros atributos funcionan
en solo alguna etiquetas como src

6. CSS

Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizado para describir la
presentación de documentos HTML

  • width / height

  • inline-size / block-size
  • margin-block | margin-inline | margin-block-start | margin-block-end | margin-inline-start | margin-inline-end
  • border-block | border-inline | border-block-start | border-block-end | border-inline-start | border-inline-end
  • padding-block | padding-inline | padding-block-start | padding-block-end | padding-inline-start | padding-inline-end
